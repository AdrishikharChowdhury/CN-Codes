# IP Info

Analyzes an IPv4 address and displays its class, IP range, subnet mask, broadcast address, and available hosts. Includes hardcoded demo addresses (Class A, B, C).

## How to Run
```bash
python main.py
```

## Example Output
```
========================================
IP Address: 192.168.100.14
Class IP: Class C
IP Range: 192.0.0.0 - 223.255.255.255
Address Mask: 192.0.0.0
Broadcast Addess: 223.255.255.255
Available Hosts  : 254
========================================
...
```
